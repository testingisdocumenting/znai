Actual Size

SVGs may have additional white space around the content and have width and height property specified inside Use if you want to use the actual content only and effectively crop the white space and will be overridden with the cropped size st0 fill #5CB8CD st1 fill #FB8A52 st2 fill #152D39 st3 fill #E8E8E8 Without there would be a gap created by hiding other parts actualSize true width height include svg with groups svg idsToReveal partC actualSize true actualSize
